# APES Pacing Planner

A standalone web application for AP Environmental Science teachers to create a personalized pacing guide based on their instructional calendar.

## Technology

- Next.js
- React
- TypeScript
- Tailwind CSS

## Status

🚧 Version 1 Development

## Current Sprint

Sprint 1 – Foundation

## Features (Version 1)

- School & Course Information
- Class Schedule
- School Calendar
- Preferences
- Personalized APES Pacing Guide
- Print
- Save PDF

## Future Versions

Additional features will be considered only after Version 1 is complete and tested with teachers.
