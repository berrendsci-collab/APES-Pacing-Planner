# APES Pacing Planner

## Purpose

The APES Pacing Planner is a standalone web application that helps AP Environmental Science teachers generate a customized pacing guide based on their instructional calendar.

---

## Version 1 Scope

### Home
- Welcome page
- Start Planning button

### Step 1 – School Information
- School name
- Teacher name (optional)
- School year
- First instructional day
- AP Exam date (teacher enters)
- Last instructional day before AP Exam
- School end date (optional)

### Step 2 – Class Schedule
- Minutes per class
- Teaching days:
  - Monday
  - Tuesday
  - Wednesday
  - Thursday
  - Friday

### Step 3 – Calendar
Teachers can mark dates as:
- Holiday
- Professional Development
- Testing
- Assembly
- Field Trip
- Weather Closure
- Half Day
- Other

### Step 4 – Preferences
- Review Days
- Flex Days

### Step 5 – Results
- Planning Summary
- Unit Table
- Year-at-a-Glance Calendar
- Print
- Save PDF

---

## Calculation Rules

- Use College Board CED midpoint percentages.
- Labs are included within unit pacing.
- Review days are user-selected.
- Flex days are user-selected.
- Display Estimated Class Periods for each unit.
