# APES Pacing Planner

## Vision

Create a web application that enables AP Environmental Science teachers to generate a customized year-long pacing guide based on:

- School calendar
- Schedule type
- APES Course and Exam Description (CED)
- AP Exam date
- Instructional preferences
- Labs
- Review sessions
- Flex days

The application should produce a pacing guide that is editable, exportable, and easy to update.
